Erstellen eines Programmes

Ein neues Programm bzw. eine neue Datei kann auf verschiedene Weise erstellt werden.

  • Im Explorer kann über das Kontext-Menü oder die Toolbar eine neue Datei erzeugt werden.
  • Im Editor kann über 'Neuer Tab' und 'Speichern unter' eine Datei gespeichert werden.

Dateiformat

Diese Anwendung verwendet ein spezielles Dateiformat (*.cnc3). Dieses Format ermöglicht es sogenannte Tags mit dem eigentlichen NC-Inhalt abzuspeichern. Diese Tags können über das Dokument-Eigenschaften Fenster im Editor erstellt werden. Über die Suchfunktion kann auch direkt nach diesen Dokument-Eigenschaften gesucht werden. Andere Text-basierte Dateiformate können importiert und exportiert werden, jedoch nicht aktiv verwaltet werden.

Wird eine neue Datei oder ein neuer Tab erzeugt und es wurde eine Standard-Vorlage festgelegt, dann wird diese mit eingefügt.
Die Programm-Eingabe kann nun über den Editor erfolgen. Vorlagen können eingefügt werden und der Inhalt des Editors
kann über die serielle Schnittstelle gesendet werden. Durch das Dokument-Eigenschaften Fenster können zusätzliche Beschreibungen für die Datei
eingegeben werden. Auch zeigt dieses Fenster die Zeit und das Datum der letzten Bearbeitung an und wann es zuletzt geöffnet wurde.

Inhalt von textbasierten Dateien kann über die Import-Funktion über die Toolbar oder durch das Kontext-Menü im Explorer importiert werden.
Auch kann durch Drag & Drop der Inhalt des Editors gesetzt werden, bzw. können cnc3 Dateien geöffnet werden.

Ebenfalls können textbasierte Dateien im Explorer über das Kontext-Menü direkt in cnc3 Dateien konvertiert werden,
mit der Option die alte Datei zu behalten oder zu löschen.
