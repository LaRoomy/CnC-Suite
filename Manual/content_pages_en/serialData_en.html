Serial Data-Exchange

Setup of the serial interface

The interface can be selected an configured using the app settings.

All devices available in the system can be enumerated with the 'Search serial devices' button. The general interface parameters are set for EPL2 controls, only the baud rate must be individually adapted to the corresponding machine.

Advanced Setup

Changes on the advanced setup parameters should only be assigned if the effect and the consequences are known. The default values are common values and should be appropriate for almost all cases.

The behavior of the transmission window and the output formatting can be adapted by using the input/output settings button.


Sending and Receiving Data

To send or receive data, the last two buttons of the main toolbar are used. Alternatively the shortcut Ctrl+T sends data and Ctrl+R starts the receiving process.

Sending:
When the transmission process is started, the output window will be opened (if monitoring is activated). The current content of the active (visible) Editor/Tab will be sent. After all data is tranmitted, the output window is closing automatically. This behavior could be disabled in the input/output settings. If the verbose output is active, additional information is printed in the transmission window. This could be helpful when encountering transmission problems.

Receiving:
When the reception process is started, the input window will be opened (if monitoring is activated). The module is waiting for incoming data. If the process is completed, the received data could be assigned to the active Editor/Tab (or discarded of course).
