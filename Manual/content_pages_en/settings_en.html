Einstellungen der Anwendung

Allgemeine Einstellungen

Hier können grundlegende Einstellungen wie Sprache und Design festgelegt werden,
sowie das Verhalten beim Programmstart und ob eine Chronik der bearbeiteten Dateien angelegt werden soll.
Wird die Sprache angepasst muss die Anwendung neu gestartet werden, damit die erforderlichen Resourcen neu geladen werden können.
Bei der Design-Auswahl stehen 3 Möglichkeiten zur Verfügung Hell, Mittel und Dunkel (Hellgrau/Dunkelgrau/Schwarz).

Beim Programmstart-Verhalten kann festgelegt werden ob die expandierten Programmpfade in der Baum-Ansicht wiederhergestellt werden sollen,
ob die letzten Tabs samt Inhalt wieder geladen werden sollen und ob Tabs die nicht gespeichert waren wiederhergestellt werden sollen.


Tab-Steuerung

Hier kann das Verhalten der Tabs angepasst werden. Wenn automatisch Speichern aktiv ist, dann wird (sofern ein Pfad festgelegt wurde) alle 15 Sekunden gespeichert.
Dadurch das eine Datei immer in einem neuen Tab geöffnet wird, kann verhindert werden das der aktuelle Tab überschrieben wird. Sollte der aktuelle Tab nicht gespeichert worden sein,
ist trotzdem eine Bestätigung erforderlich. Ungespeicherter Inhalt wird nicht überschrieben.
Sollten die Dokument-Eigenschaften nicht benutzt werden kann hier festgelegt werden das diese ausgeblendet werden.

Die Beschreibung der Dokument-Eigenschaften können an die eigenen Bedürfnisse angepasst werden.
Wenn eine Standard-Vorlage gespeichert wurde kann festgelegt werden ob sie bei einem neuen Tab mit eingefügt wird.
Zuletzt kann das Format des Zeilenendes der Export-Funktion festgelegt werden.


Explorer Einstellungen

Wenn eine Datei über den Öffnen-Button in der oberen Toolbar geöffnet wurde,
kann hier bestimmt werden ob der Pfad in der Baum-Ansicht des Explorers expandiert wird. Vorraussetzung ist sich der Pfad innerhalb der Struktur des Stammordners befindet.
Sollte eine neue Datei im Explorer erzeugt werden, kann hier festgelegt werden ob diese auch direkt als Tab geöffnet wird.
Die dritte Checkbox legt fest ob die Dokument-Eigenschaften als Tooltips angezeigt werden wenn mit der Maus darauf gezeigt wird.

Zudem kann hier das Verhalten des Suchfensters angepasst werden und die Elemente nach denen gesucht wird können hier festgelegt werden.
